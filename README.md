# Expense Tracker (Flask + SQLite)

A simple full-stack expense tracker built with Python and Flask.

## Features
- Add expenses
- View all expenses
- Delete expenses
- Persistent storage using SQLite

## Tech Stack
- Python
- Flask
- SQLite
- HTML/CSS

## How to Run

```bash
git clone <your-repo-url>
cd expense-tracker
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
